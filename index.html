What is GEO and why does it matter for Silverpush?

GEO (Generative Engine Optimisation) is the practice of structuring content so it gets cited and surfaced in AI-generated responses. As ad buyers turn to ChatGPT, Perplexity, and Gemini asking "What is contextual advertising?" or "Best cookieless targeting solution?", Silverpush needs its content to be the authoritative source those AI engines cite.

Unlike traditional SEO (10 blue links), GEO focuses on being cited as a source inside AI summaries — driving brand authority among decision-makers actively researching solutions.

⚙️ API Settings

📊 Google Analytics 4
Your GA4 property Measurement ID. Found in GA4 → Admin → Data Streams → Web stream → Measurement ID.
Create a restricted API key in Google Cloud Console:
1. Enable Google Analytics Data API
2. Restrict key to your domain: https://silverpush.co/*
3. Restrict to Analytics Data API only
🔍 Google Search Console
Enable Google Search Console API in Cloud Console and add it to the same restricted key. Your site must be verified in Search Console first.
🔢 GA4 Numeric Property ID
The numeric Property ID from GA4 → Admin → Property Settings (not the G-XXXXXX Measurement ID — this is the plain number used in API calls).
🔒 Security model
API keys are stored only in your browser's localStorage under the key sp_keys. They are never embedded in the HTML file, never transmitted to any server, and never leave your device. Clearing browser data will remove them — you'll need to re-enter them.
